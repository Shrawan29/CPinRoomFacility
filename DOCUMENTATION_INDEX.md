## üéì Key Concepts

### Understanding QR Codes
- Each QR contains a unique URL
- URL includes token and room number
- Token is single-use only
- QR codes placed in each room

### Understanding Authentication
- 3-step process: QR ‚Üí Phone ‚Üí OTP
- Each step adds security
- Backend validates at each step
- Session created after all steps pass

### Understanding Data Flow
- Guest action ‚Üí Frontend ‚Üí API ‚Üí Backend
- Backend validates ‚Üí Database ‚Üí Response
- Frontend updates UI ‚Üí Redirect or error

### Understanding Security
- Multiple validation layers
- Single-use tokens
- Phone verification
- OTP authentication
- Session management
